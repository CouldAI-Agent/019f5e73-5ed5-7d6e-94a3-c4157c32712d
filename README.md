# Aplikasi Presentasi Flutter

Aplikasi ini adalah contoh sederhana bagaimana membangun antarmuka slide presentasi (mirip dengan PowerPoint atau Keynote) menggunakan Flutter.

## Fitur Utama
- **Navigasi Slide:** Berpindah antar slide dengan mudah menggunakan gestur geser (swipe) atau tombol panah navigasi.
- **Responsif:** Tampilan dioptimalkan baik untuk ukuran layar besar (Desktop/Web) maupun layar kecil (Ponsel).
- **Berbasis Widget:** Tidak seperti presentasi tradisional, karena ini adalah aplikasi Flutter, setiap slide dapat diisi dengan animasi kompleks, formulir interaktif, panggilan API, dan fitur aplikasi nyata lainnya.
- **Beberapa Contoh Slide:** Dilengkapi dengan slide judul, slide fitur (poin-poin), slide konten dengan layout fleksibel, dan slide penutup yang memiliki interaksi tombol.

## Menjalankan Aplikasi
Aplikasi ini dapat dijalankan di semua platform yang didukung Flutter (Web, iOS, Android, macOS, Windows, Linux).

Untuk menjalankan aplikasi ini secara lokal:
1. Pastikan Flutter sudah terinstal di perangkat Anda.
2. Jalankan perintah untuk mengunduh dependensi:
   ```bash
   flutter pub get
   ```
3. Jalankan aplikasi:
   ```bash
   flutter run
   ```
